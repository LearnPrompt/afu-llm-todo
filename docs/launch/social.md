# 社媒发布文案

## GitHub 发布帖

我做了一个给 Obsidian Agent 用的 Skill。

名字叫管家阿福，英文是 Afu · LLM Todo。

它不是又一个 Todo App。

它做的是前面那一步，把 Obsidian 收件箱里的资料先整理成 Wiki，再从 Wiki 里生成本周能推进的待办卡。

```text
Inbox -> Wiki -> Todo Card -> Calendar
```

如果你的 Obsidian 里也堆了很多网页、帖子、论文、灵感和项目记录，可以试一下。

```bash
npx skills add LearnPrompt/afu-llm-todo
```

Repo: https://github.com/LearnPrompt/afu-llm-todo

## ClawHub 发布帖

谁不想拥有一个自己的管家阿福。

不是替你思考人生的那种，也不是把你的 Obsidian 搬去另一个数据库。

它只是站在 Vault 门口，把收件箱里的东西整理成 Wiki，再把 Wiki 里已经成型的判断变成待办卡。

然后你再排进日历。

这件事很小，但我自己的感受是，挺救命的。

## X / Twitter

Built Afu, an LLM Todo Skill for Obsidian agents.

Not another todo app.

It turns an Obsidian inbox into a living Markdown wiki, then into todo cards for this week.

Inbox -> Wiki -> Todo Card -> Calendar.

Local-first. Markdown stays yours.
