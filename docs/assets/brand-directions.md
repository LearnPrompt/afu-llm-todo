# 管家阿福视觉方向

当前主视觉为 `docs/assets/afu-hero.png`：使用最终生成图，卡通化人物双手接住紫色黑曜石晶体。

视觉不做真人照片复刻，不复制 Obsidian 官方 logo，不使用 Batman 官方视觉。只借用“管家、晶体、Wiki 网络、行动卡、日历”这些抽象元素。

## 方向 1：Afu Butler + Obsidian Crystal

文件：`docs/assets/afu-logo-butler-crystal.svg`

含义：

- 托盘代表管家阿福。
- 晶体代表 Obsidian / Markdown Vault。
- 三张卡片代表行动卡。

适合用作 README 首图和 ClawHub 头像。

## 方向 2：LLM Wiki Graph + Calendar

文件：`docs/assets/afu-logo-wiki-calendar.svg`

含义：

- 节点网络代表 LLM Wiki。
- 下方日历格代表周排期。
- 连线代表 Wiki 到 Todo 的转化。

适合用作技术解释图。

## 方向 3：Inbox Alchemy -> Todo Cards

文件：`docs/assets/afu-logo-inbox-alchemy.svg`

含义：

- 左侧收件箱代表粗素材。
- 中间星点代表编译和判断。
- 右侧行动卡代表本周可执行事项。

适合社媒配图。
